¿Qué es Internet?

Internet es una red global de computadoras interconectadas que permite el intercambio de información y servicios, como correo electrónico, navegación web, redes sociales, entre otros. Fue desarrollada inicialmente para fines militares y académicos, pero hoy es una herramienta esencial en la vida diaria.

Tipos de Proveedores de Internet y nombres

Los proveedores de servicios de Internet (ISP) pueden ser:

  • Proveedores de Banda Ancha: Ofrecen conexiones rápidas (ej. fibra óptica, ADSL).
  • Proveedores Móviles: Ofrecen Internet mediante redes celulares (4G, 5G).
  • Proveedores Satelitales: Usan satélites para ofrecer servicio en zonas rurales.

Ejemplos de ISPs: Movistar, Claro, Vodafone, AT&T, HughesNet.

¿Qué es WWW?

WWW significa World Wide Web, es un sistema de documentos interconectados a través de hipervínculos y accesibles mediante navegadores. Fue inventada por Tim Berners-Lee en 1989 y es lo que comúnmente usamos para navegar sitios web.

¿Qué es carpeta local y carpeta remota?

Carpeta local: Es la que está en tu computadora personal.

Carpeta remota: Es la que se encuentra en un servidor web, a la cual se accede mediante protocolos como FTP para alojar sitios web.

¿Qué es FTP?

FTP (File Transfer Protocol) es un protocolo que permite transferir archivos entre computadoras a través de Internet. Se utiliza comúnmente para subir archivos a servidores web.

¿Qué es experiencia de usuario?

Una carpeta local es un directorio en tu ordenador que almacena archivos directamente en tu disco duro, mientras que una carpeta remota se encuentra en otro dispositivo o servidor, como un servidor web o en la nube, y a la que se accede a través de una red o internet.
